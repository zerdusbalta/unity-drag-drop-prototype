# Unity Drag & Drop Prototype

A simple Unity drag-and-drop prototype demonstrating interaction logic, snapping, and drop validation.

## Features
- Drag-and-drop interaction
- Drop zone validation
- Overlap threshold logic
- Smooth snap animation
- Success text feedback
- Reset to start on incorrect drop

## Tech Stack
- Unity (2D)
- C#

## How to Play
1. Click and drag the white square
2. Move it into the red drop zone
3. Release to snap into place


## Goal
This project was built to practice interaction logic and create a small playable prototype inspired by mobile game interaction systems.
